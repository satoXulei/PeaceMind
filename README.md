## 🌱 为什么做这件事？

### 我们生活在一个不容易平静的时代

你是否也经历过这样的时刻——

深夜刷手机，明明很累却舍不得放下，怕一停下来就被焦虑吞没；  
打开社交媒体，看到的全是“月入十万”“自律逆袭”，对比之下觉得自己一无是处；  
遇到挫折想找人聊聊，却发现大家都在赶路，没人有空听你慢慢说；  
甚至，你连自己到底在不安什么，都说不清楚。

### 尤其是，来自工作的压力

你有没有在上班路上，坐在车里迟迟不想熄火？  
有没有在周一凌晨醒来，心慌得再也睡不着？  

工作本应是生活的一部分，可很多时候它压得人喘不过气——  
KPI、汇报、人际关系、裁员焦虑……白天被会议和消息淹没，晚上回家只想瘫着，却发现还有一堆家务和孩子在等着。

更让人无力的是：**我们很难说“我不干了”**。

因为：

- 房贷每个月都要还，银行不会因为你的情绪而延期
- 车贷、信用卡账单，一张都不会少
- 孩子的学费、兴趣班、奶粉钱，都是实打实的开支
- 父母年纪大了，万一有个病痛，需要的不仅是陪伴，还有钱

于是，我们咬着牙上班，忍着委屈加班，不断告诉自己：“都是为了生活。”

但慢慢你会发现——  
**用透支内心换来的钱，最终也买不回内心的安宁。**

工作的意义当然重要，但当我们把自己的全部价值绑在工位上、绑在薪水数字上时，任何一次挫折都可能变成一场内心的崩塌。

这不是你脆弱，这是这个时代太多人正在承受的重量。

### 书，曾是我最低成本的治愈方式

在我最迷茫的那段日子，我没有钱看心理咨询师，也不知道该向谁开口。  
但有一本书，让我第一次觉得“原来有人懂我”；  
有一句话，在我想要放弃自己的时候，轻轻拉了我一把。

从那之后，我开始有意识地去读心理治愈类的书——  
不是为了成为专家，只是为了**让自己好过一点**。

我发现：  
一本对的书的出现在恰当的时间，就像一个沉默却坚定的朋友。  
它不会评判你，不会催促你，只是安静地告诉你——“你不孤单，这条路有人走过，你也可以。”

### 所以，我们想做这件事

**PeaceMind** 不是心理诊所，不是学术数据库，也不是另一个制造焦虑的“成长社群”。  
它只是一个简单到近乎笨拙的想法：

> 把那些曾经治愈过我们的书，整理出来，分享给此刻需要的人。

我们不收钱，不卖课，没有复杂的算法。  
只有一份干净的书单，和每本书背后一段真诚的推荐理由。

我们希望：  
- 当你被工作压得喘不过气，打开 PeaceMind，能找到一本陪你度过通勤路的书。  
- 当你因为房贷、养家的压力而失眠时，书里的某句话能让你松一口气。  
- 当你在职场中感到迷失，想找回“我到底为什么而活”的答案时，这里有一盏小小的灯。

### 真正的平静不在远方

安静不是没有声音，而是在喧闹中依然能听见自己。  
治愈不是没有伤痛，而是学会与伤痛共处。

PeaceMind 改变不了你的工资单，也替你还不了房贷。  
但如果它能让你在翻几页书的片刻里，稍微松一口气——  
那就已经值得了。

**心安，即是归处。**  
欢迎你来，也欢迎你留下点什么。
